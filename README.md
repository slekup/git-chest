# Git Chest

Store and sync repository metadata (data optional) for offline browsing, analysis, and updates.

## Supported Git Hosts

- [ ] GitHub
- [ ] GitLab
- [ ] Gitea

## Run in development

```sh
git clone https://github.com/slekup/git-chest
cargo tauri dev
```
